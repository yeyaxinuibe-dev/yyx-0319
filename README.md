🍽️ FoodiePass - 每日一味
每周一道新风味，吃遍全世界

一个有趣的一周食谱应用，带你探索15个国家的美食文化，每天一道特色菜！

Next.js
TypeScript
Tailwind CSS

✨ 特色
🌍 15个国家美食 - 西班牙、日本、意大利、法国、泰国、墨西哥、印度、韩国、巴西、希腊、越南、土耳其、摩洛哥、秘鲁、中国
🎭 MBTI性格匹配 - 每个国家对应一个MBTI人格，配有独特的标语和人物形象
🍜 每日精选 - 每周7天，每天一道精心挑选的特色菜
🎲 随机探索 - 点击换一个国家，开启新的美食之旅
🚀 快速开始
环境要求
Node.js 18+
npm 或 yarn
安装
# 克隆项目
git clone https://github.com/yeyaxinuibe-dev/foodepass-weekly-menu.git

# 进入目录
cd foodepass-weekly-menu

# 安装依赖
npm install

# 启动开发服务器
npm run dev
打开浏览器访问 http://localhost:3000 即可查看！
